

يهدف هذا المشروع إلى تنفيذ مفهوم المخزن الدائري (Circular Buffer) باستخدام لغة C، وهو هيكل بيانات يُستخدم لتخزين البيانات بطريقة دائرية دون الحاجة إلى تحريك العناصر داخل المصفوفة.

# مكونات المخزن

يعتمد المخزن الدائري على:

head → مؤشر القراءة

tail → مؤشر الكتابة

count → عدد العناصر

size → حجم المخزن

# العمليات المدعومة

تهيئة المخزن

إدخال عنصر (Enqueue)

قراءة عنصر (Dequeue)

التحقق من الامتلاء (Overflow)

التحقق من الفراغ (Underflow)

#آلية عمل البرنامج

- يدخل المستخدم اسمه
- تُضاف العبارة "CE-ESY" إلى الاسم
- يُخزَّن كل حرف داخل المخزن الدائري
- تُقرأ البيانات مرة أخرى وتُعرض على الشاشة
- يتم التحقق من أن المخزن فارغ بعد القراءة

## التعامل مع الامتلاء والفراغ
- إذا كان المخزن ممتلئاً ← تظهر رسالة Overflow
- إذا كان المخزن فارغاً ← تظهر رسالة Underflow



#حالات الاختبار

الحالة الأولى: مخزن صغير

حجم المخزن أقل من طول النص

النتيجة:

ظهور رسالة "المخزن ممتلئ"

تخزين جزئي للنص

الحالة الثانية: مخزن كبير

حجم المخزن أكبر من طول النص

النتيجة:

تخزين كامل للنص

قراءة صحيحة

المخزن يصبح فارغًا ✔
